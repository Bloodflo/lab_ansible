# DevOps Практическая работа П04: Деплой веб-сервера с помощью Ansible

## 📝 Описание
Данная работа демонстрирует автоматизацию развертывания веб-сервера Nginx с использованием Ansible.

## 🎯 Цель работы
- Установка и настройка Ansible
- Создание inventory файла
- Написание Ansible playbook
- Автоматизация установки Nginx
- Деплой статического контента

## 📁 Структура репозитория
ansible-p04/
├── README.md # Описание проекта
├── playbooks/
│ └── playbook.yml # Основной playbook
├── inventory/
│ └── inventory.ini # Файл инвентаризации хостов
├── files/
│ └── index.html # Шаблон индексной страницы
└── screenshots/ # Скриншоты выполнения
